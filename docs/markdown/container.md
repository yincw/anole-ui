# 容器

## 自定义容器

:::info
这是一条普通信息
:::

:::success
这是一条成功信息
:::

:::warning
这是一条警告信息
:::

:::error
这是一条错误信息
:::

:::info{title=自定义标题}
这是一条普通信息
:::

## 警告提示

<Alert>
  Text
</Alert>

<Alert type="info">
  Info Text
</Alert>

<Alert type="success" showIcon >
  Success Text
</Alert>

<Alert type="warning" showIcon>
  Warning Text
</Alert>

<Alert type="error" showIcon closable >
  Error Text
</Alert>
